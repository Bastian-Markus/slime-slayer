<h1>Slime Slayer</h1>

<p>This is my very first game, made in the tiny engine <a href="https://tic80.com/">TIC-80</a>.
The engine intentionally comes with severe technical limitations to mimic retro style game development.
</p>

<h3>Engine Limits</h3>
<ul>
  <li><strong>Screen Resolution:</strong> 240 × 136 px</li>
  <li><strong>Code Limit:</strong> Single file, maximum size of 64 KB</li>
  <li><strong>Graphics:</strong> 512 tiles (8×8 px) for sprites and VFX</li>
  <li><strong>Audio:</strong> 4 channels</li>
  <li>and more...</li>
</ul>
